# Lingva Solidare

A proof-of-concept marketing site for a student initiative called Lingva Solidare.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - App Router directory
  - `layout.tsx` - Root layout with navigation and footer
  - `page.tsx` - Home page
  - `components/` - Reusable components
    - `Navigation.tsx` - Top navigation bar
    - `Footer.tsx` - Site footer
  - `globals.css` - Global styles with Tailwind

## Design

- Clean, modern design with dark text on light background
- Accent color: Cyan/Teal (#0891b2) for a modern, international feel
- Academic + activist + cultural aesthetic
- Responsive design for desktop and mobile
- Centered content container with max-width of 1024px
